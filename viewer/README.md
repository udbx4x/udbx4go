# udbx4go-viewer

UDBX 文件图形化查看器 - 基于 Go 和 Fyne 的跨平台 GUI 工具。

## 功能特性

- **文件浏览**: 打开本地 .udbx 文件
- **数据集列表**: 左侧树形展示所有数据集（带类型图标）
- **数据查看**: 右侧表格展示数据记录
- **分页展示**: 每页 100 条，支持翻页导航
- **类型支持**: Tabular, Point, Line, Region, PointZ, LineZ, RegionZ

## 技术栈

- **GUI 框架**: [Fyne](https://fyne.io/) v2.5.0 (纯 Go，无 CGO)
- **核心库**: udbx4go
- **Go 版本**: 1.22+

## 快速开始

### 安装依赖

```bash
cd cmd/udbx4go-viewer
go mod tidy
```

### 构建

```bash
go build -o udbx4go-viewer
```

### 运行

```bash
./udbx4go-viewer
```

## 使用说明

1. 点击 **文件 → 打开文件...**
2. 选择 .udbx 文件
3. 在左侧数据集列表中点击要查看的数据集
4. 右侧表格显示数据记录
5. 使用底部按钮翻页浏览

## 功能特性

- ✅ 文件浏览：打开本地 .udbx 文件
- ✅ 数据集列表：左侧树形展示所有数据集
- ✅ 类型图标：根据数据集类型显示不同图标

- ✅ 分页展示：每页 100 条，支持首页/末页/上一页/下一页
- ✅ 状态栏：显示当前操作状态
- ✅ 资源管理：自动释放数据库连接

## 图标说明

| 图标 | 颜色 | 类型 |
|------|------|------|
| ● | 蓝色 | Point (2D) |
| ● | 深蓝色 | PointZ (3D) |
| ▬ | 绿色 | Line (2D) |
| ▬ | 深绿色 | LineZ (3D) |
| ▲ | 橙色 | Region (2D) |
| ▲ | 深橙色 | RegionZ (3D) |
| ☰ | 灰色 | Tabular |
| ? | 黄色 | 未知类型 |
