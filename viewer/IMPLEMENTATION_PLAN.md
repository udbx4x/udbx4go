# udbx4go-viewer 实施计划

## 阶段一：项目初始化（1-2 小时）

### 1.1 创建目录结构
```bash
mkdir -p cmd/udbx4go-viewer
cd cmd/udbx4go-viewer
go mod init github.com/udbx4x/udbx4go/cmd/udbx4go-viewer
```

### 1.2 添加依赖
```bash
go get fyne.io/fyne/v2@v2.5.0
go get github.com/udbx4x/udbx4go@latest
```

### 1.3 验证构建
```bash
go build -o udbx4go-viewer
```

**完成标准：** 空项目成功编译，无报错。

---

## 阶段二：基础窗口与菜单（2-3 小时）

### 2.1 实现 main.go
- 创建 Fyne 应用实例
- 设置窗口标题 "UDBX Viewer"
- 设置初始窗口大小 1000x650

### 2.2 实现 app.go
- 创建主窗口布局（左右分割）
- 添加菜单栏（文件、帮助）
- 添加状态栏

### 2.3 实现文件打开功能
- 绑定菜单项 "打开文件..."
- 调用 dialog.NewFileOpen()
- 文件过滤：*.udbx

**完成标准：** 能打开窗口，点击菜单弹出文件选择对话框。

---

## 阶段三：数据集树形列表（3-4 小时）

### 3.1 实现 icons.go
- 为每种 DatasetKind 创建彩色图标
- 使用 canvas.NewRectangle() + container.NewStack()
- 参考 udbx4j 的颜色方案

### 3.2 实现 tree.go
- 创建 DatasetTree 组件
- 绑定 UDBX 文件打开事件
- 使用 widget.Tree 展示数据集列表

### 3.3 实现数据集节点渲染
- 显示图标 + 名称 + 记录数
- 不同类型的数据集显示不同图标

**完成标准：** 打开文件后左侧显示数据集树，带图标区分类型。

---

## 阶段四：数据表格展示（4-5 小时）

### 4.1 实现 dataset.go
- 封装数据集加载逻辑
- 实现 LoadDataset() 方法
- 处理不同类型数据集

### 4.2 实现 table.go
- 创建 DatasetTable 组件
- 动态生成表格列
- 处理 SmID、Geometry、属性字段

### 4.3 实现数据加载
- 点击树节点加载对应数据集
- 使用 goroutine 异步加载
- 加载时显示进度指示

**完成标准：** 点击数据集能在右侧显示数据表格。

---

## 阶段五：分页功能（2-3 小时）

### 5.1 实现分页组件
- 底部添加分页按钮：<< < 第 X/Y 页 > >>
- 每页显示 100 条记录

### 5.2 实现分页逻辑
- 计算总页数
- 切换页码时重新加载数据
- 使用 QueryOptions 的 Limit/Offset

### 5.3 优化用户体验
- 页码跳转输入框
- 首页/末页快速跳转

**完成标准：** 大数据集分页正常，切换流畅。

## 风险与应对

| 风险 | 应对策略 |
|------|---------|
| Fyne 学习成本 | 参考官方示例，渐进式实现 |
| 大数据集内存问题 | 严格分页，控制每页数量 |
| 表格性能问题 | 使用虚拟列表，只渲染可见行 |
| 跨平台兼容 | 使用纯 Go 方案，避免平台特定代码 |
